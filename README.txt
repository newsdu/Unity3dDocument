------------------------- How to use ------------------------------------

1. Select word or position cursor for ask in Text Editor.

2. Choose menu you want.

  - Unity3d Local Document (Ctrl+F1, Ctrl+F1)
  - Unity3d Web Document (Ctrl+F2)
  - Google Search (Shift+F1)
  - Naver Search (Shift+F2)

3. See the result easily!

-------------------------------------------------------------------------

This can configure several option. To set unity3d installed path, 
webrowser executable path, google search prefix word, naver search prefix word. 
It's saved on your document folder/AppData/Local/Unity3dDocument directory.
Option file name is "Unity3dDocumentOption.xml".
